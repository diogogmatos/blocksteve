resolveJogo :: Int -> Jogo -> Maybe [Movimento] #

A função ’resolveJogo’ calcula o menor número de movimentos necessários para resolver um jogo.

!! : Função inconcluída - funciona apenas para mapas que não contêm caixas.

resolveSemCaixas :: Int -> [(Peca, Coordenadas)] -> Jogador -> [Movimento] #

A função ’resolveSemCaixas’ calcula o menor número de movimentos para resolver um jogo sem caixas.

Para isso, a função verifica as coordenadas da Porta do mapa e, dependendo da sua posição em relação ao jogador (à direita ou à esquerda), irá efetuar movimentos nessa direção até o jogador chegar à porta, ou seja, até o jogador ter coordenadas iguais às coordenadas da porta.

Sendo uma função recursiva, para cada movimento irá efetuar um conjunto de avaliações de modo a saber qual é o próximo passo. Essas avaliações são:

Verifica se existe ou não um Bloco em frente ao jogador:

  • Se existe, efetua o movimento Trepar para subir o bloco;
  • Caso contrário, efetua o movimento AndarDireita ou AndarEsquerda, dependendo da orientação.

Nota: Quando o jogador carrega uma caixa, verifica também se existe espaço para transportar a caixa, caso não exista retorna Nothing.
